# AI-BASED-SOCIAL-INCLUSION-TOOL-for-ELDERLY-PROPLE
The AI-Based Social Inclusion Tool for Elderly People (Wellness Companion) is a smart application that helps senior citizens manage their daily health, reminders, and activities using AI. It provides features like a chatbot for assistance, health and water-intake reminders, and entertainment suggestions to reduce loneliness and improve well-being. The system is easy to use and supports independence, safety, and better quality of life for elderly users. Overall, it connects technology with elderly care to make daily living more comfortable and secure.
